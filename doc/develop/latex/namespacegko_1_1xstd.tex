constexpr helper which checks if lhs $>$ rhs. 

This is helpful within template declarations since \char`\"{}$>$\char`\"{} cannot directly be used. Note that std\+::greater is available as constexpr only since C++14. This does not implement all the functionality of C++14, only what is needed so far.
