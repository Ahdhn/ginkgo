\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename Value\+Type, size\+\_\+type Dimensionality$>$\newline
class gko\+::accessor\+::block\+\_\+col\+\_\+major$<$ Value\+Type, Dimensionality $>$}

A bridge between a range and a block-\/column-\/major memory layout. 

Only the innermost two dimensions are regarded as defining a column-\/major matrix, and the rest of the dimensions are treated identically to \mbox{\hyperlink{classgko_1_1accessor_1_1row__major}{row\+\_\+major}}.

You should not try to explicitly create an instance of this accessor. Instead, supply it as a template parameter to a range, and pass the constructor parameters for this class to the range (it will forward it to this class).
