template<typename ConcreteBlockJacobiFactory>
void gko::preconditioner::BasicBlockJacobiFactory< ConcreteBlockJacobiFactory >::set_block_pointers ( const Array< index_type > &  block_pointers)

Sets the array of block pointer which will be used to determine the blocks for the matrix.

Parameters
block_pointersThe array of block pointer, the value at position i of the array should be set to the row where the i-th block starts. In addition, the last value of the array should be set to the number of rows of the matrix. Thus, the total number of blocks is block_pointers.get_num_elems() - 1.
