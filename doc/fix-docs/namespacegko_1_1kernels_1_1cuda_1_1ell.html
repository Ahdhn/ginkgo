Typedef Documentation

using gko::kernels::cuda::ell::compiled_kernels = typedef syn::value_list<int, 0, 1, 2, 4, 8, 16, 32>

A compile-time list of sub-warp sizes for which the spmv kernels should be compiled.

0 is a special case where it uses a sub-warp size of 32 in combination with atomic_adds.
