Example programs

Here you can find Examples programs that demonstrate the usage of Ginkgo.

Some examples are built on another and some are stand-alone and demonstrate a concept of Ginkgo, which can be used in your own code.

You can browse the available example programs

  1. as a graph that shows how example programs build upon each other.
  2. as a list that provides a short synopsis of each program.
  3. or grouped by topic.

The programs are in the examples/ directory of your local Ginkgo installation. To compile the examples during the compilation of the library, you can pass the -DGINKGO_BUILD_EXAMPLES = ON to the cmake command. It is ON by default. After compiling the library itself, if you go into one of the example directories, you can configure the program by typing cmake ., build it via make and run it using make run.
